# CineSearch - Movie Explorer

CineSearch is a responsive movie explorer web application built using HTML, CSS, and JavaScript. It allows users to search for movies, view movie cards, and open detailed movie information using the OMDb API.

## Features

* Search movies by title
* Fetch movie data from OMDb API
* Display movie posters, title, year, and type
* View full movie details on card click
* Show movie plot, genre, runtime, rating, director, and actors
* Back to results functionality
* Loading message while fetching data
* No movies found message
* Placeholder image for movies without posters
* Responsive design for desktop and mobile screens

## Technologies Used

* HTML
* CSS
* JavaScript
* OMDb API
* Fetch API
* Async/Await
* DOM Manipulation

## How It Works

1. The user enters a movie name in the search box.
2. The app sends a request to the OMDb API.
3. Matching movies are displayed as cards.
4. When the user clicks a movie card, the app fetches full movie details.
5. The detailed movie page is displayed with a back button to return to search results.
